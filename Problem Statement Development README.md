**Problem**

    Students lack a centralized place to access e-books provided by faculty or institutions.

**Solution**

Create a web-based system where:

    1.Admin can upload, edit, delete e-books
    
    2.Users can view, download, search, and review books

**Target Users**

    Students, Librarians, Faculty

**Value Proposition**

    1.Quick access to digital books
    
    2.Well-organized categories
    
    3.User-friendly interface
    
    4.Review and feedback system
